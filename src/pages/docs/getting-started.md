## Introduction

Welcome to your first markdown document! This guide will help you understand the basics of markdown syntax and how to create beautiful documentation.

## What is Markdown?

Markdown is a lightweight markup language that allows you to write using an easy-to-read, easy-to-write plain text format. It's perfect for:

- Documentation
- README files
- Blog posts
- Notes and wikis

## Basic Syntax

### Headers

You can create headers using the `#` symbol:

```markdown
# H1 Header
## H2 Header
### H3 Header
```

### Emphasis

You can make text **bold** or *italic*:

- **Bold text** using `**bold**` or `__bold__`
- *Italic text* using `*italic*` or `_italic_`

### Lists

Create unordered lists with `-`, `*`, or `+`:

- Item 1
- Item 2
  - Nested item
  - Another nested item
- Item 3

Or ordered lists with numbers:

1. First item
2. Second item
3. Third item

### Links and Images

Create links with `[text](url)`:

[Visit Astro](https://astro.build)

### Code

Inline code with backticks: `const example = true;`

Code blocks with triple backticks:

```javascript
function hello() {
  console.log("Hello, World!");
}
```

### Blockquotes

> This is a blockquote.
> It can span multiple lines.
